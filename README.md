Food Management System (C++)

A console-based Food Management System developed in C++ using Object-Oriented Programming and Data Structures concepts. The project simulates a restaurant food ordering and inventory management system where customers can browse menu items, place orders, and make payments, while administrators can manage food inventory, monitor sales, and track order history.

The system uses Binary Search Trees (BST) for efficient food item storage and searching, and Linked Lists for maintaining order history and admin activity logs. It demonstrates practical implementation of inventory handling, recursion, authentication, sales tracking, and dynamic data management.
